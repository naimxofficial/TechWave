# 🎙️ TechWave

> **Your daily dose of tech insights, productivity hacks, and personal growth.**

TechWave is a modern podcast landing page for a tech-focused show hosted by Alex Kumar. The site showcases featured episodes, highlights why listeners choose TechWave, and lets visitors subscribe or listen directly on Spotify — all wrapped in a sleek dark-purple aesthetic with neon green accents.

🌐 **Live Site:** *https://naimxofficial.github.io/TechWave/*

---

## ✨ Features

| Feature | Details |
|--------|---------|
| 🏠 **Hero Section** | Animated microphone, "New" badge, Listen on Spotify & Subscribe buttons |
| 🎧 **Listen on Spotify** | Direct Spotify CTA button in the hero banner |
| 📊 **Stats Section** | Key numbers — 150K+ Monthly Listeners, 200+ Episodes, 4.9★ Rating, 50+ Industry Experts |
| ✅ **Why Choose TechWave** | 5-feature bento-style grid — Premium Audio, Mobile Friendly, Global Community, Exclusive Interviews & Rich Resources |
| 🎬 **Featured Episodes** | 3 embedded YouTube episodes with title, description & duration |
| 👤 **Meet The Host** | Host card for Alex Kumar with bio and LinkedIn, X, Instagram & YouTube social links |
| 📡 **Footer Platform Links** | Quick links to Spotify, Apple Podcast, YouTube & Twitter |
| 📱 **Hamburger Menu** | Mobile-friendly collapsible nav menu for small screens |
| ✨ **Microphone Fade Animation** | CSS keyframe animation on the hero microphone image |
| 📱 **Fully Responsive** | Optimized layouts for mobile, tablet, and desktop with media queries |
| 🎨 **Dark Neon UI** | Deep purple background with neon green accents and gradient buttons |
